Ribbit Romance: Vol VII

2024

About the Project

Ribbit Romance is a 2.5D side-scrolling platformer game made in Godot for the 2024 Global Game Jam. You play as a prince who, after an unfortunate run-in with a witch, has been turned into a frog. You must ascend the witch's tower defeating her rat henchmen and seducing the character at the top of each level to progress in search of true love's kiss. The game was created in under 48 hours within a team of nine, with myself taking lead in game design and team organisation.

Key Contributions

  • Game Design: Led the design process from start to finish including creating and maintaining the Game Design Document.
  • Project Management: Organised out teams Agile workflow using Trello.
  • Gameplay Programming: Helped create major gameplay features and integrate the games narrative using the plugin Fungus.
