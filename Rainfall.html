Rainfall

2024

Background info

Rainfall is a project that I worked on as part of a level design module in university. During the module we developed design specific skills such as level design and construction, level pacing, the importance of audio and lighting in level design, and the aid of modular kits to aid in level design. I also used this module to develop skills in the software World Machine that I used to create the custom terrain.

About the game

Rainfall is an atmospheric walking simulator that aims to elicit an emotional response through the use of visual and audio effects via a dynamic weather system. The project also features custom made terrain and other mechanics disguised as weather effects such as wind that pushes the player through the level or fog that teleports the player back to the start.

Skills Gained

I chose to complete this project in Unreal Engine due to the limited timeframe and how the module put more of an emphasis on level creation which meant I could work in Unreal's Blueprint system as the programming needed for the project wasn't super complicated. Before I started work in Unreal however, I had to learn the World Machine software. World Machine is a program that allows you the create custom terrain from nodes along with texture and normal maps. Learning World Machine was a little difficult due most of the online tutorials being outdated, but once I understood how to use the software, the output freedom to create terrain exactly how I wanted without needed to actually model it was very useful.
