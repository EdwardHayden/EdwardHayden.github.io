About the Project

Go North is a project I created to help me understand Unity Netcode for Game Objects and how to create networked multiplayer games. You play as an adventurer exploring an underground keep, solving puzzles and discovering artefacts that can help you in your journey through an interactive environment. The game itself is a fairly simple text-based-adventure/puzzle game but it allows for multiple people to join you in your keep as you play and help you solve the puzzles.

Key Contributions

  • Networked Multiplayer: Netcode for Game Objects is used to facilitate multiple players joining the same game.
  • Level Design: I created the layout of the keep and how the puzzles interact with the environment.
  • Game Design: I designed the puzzles, the gameplay loop, and the player experience.
  • UI Programming: Programmed all the UI including a key word detection feature to change the font of commands and objects.
