Runeguard

2022

About the Project

Runeguard is a game I created before attending university in 2022, I was the only developer working on it, using an asset pack I purchased for the artwork. You play as an adventurer trying to escape a castle, where you must battle your way past enemies to pull a lever that leads you to fight a boss. The game features a Fog of War style lighting system, a unique combined stamina turn based mechanic, an inventory system, a levelling up system, and two sections to play through.

Key Contributions

  • Combined Stamina/Turn Based Mechanic: Designed and implemented a mechanic that uses the stamina bar to manage everything a player can do in a turn.
  • Fog of War Mechanic: Created a fog of war like lighting system that hides enemies on the screen until they are in direct line of sight.
  • UI Design/Programming: Designed and programmed all UI in the game.
  • Debug Overlay: Created a debug overlay for the game to help during development that shows relevant information for specific game objects on screen.
