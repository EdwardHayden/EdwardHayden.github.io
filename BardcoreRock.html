Bardcore Rock

2023

About the Project

Barcore Rock is a project I created for a university module on Custom Game Controllers. I have always enjoyed working with electronics so for this project I created two separate controllers, a Keytar made from an Arduino and an old kids keyboard toy, and a Dance Game Pad that you can stand on which I took apart and also connected to an arduino to handle the inputs. The game itself is a very simple infinite runner where you can use the dance pad to swap between three horizontal lanes, and you can press keys on the Keytar to damage the enemies appreaching down the hallway.

Key Contributions

  • Physcial Computing: I used micro-controllers for input handling, and re-purposed old electronic toys to create two physical game controllers.
  • Custom Game Controllers: I used a plugin in Unity to allow data transfer through the serial bus and translate that into inputs for a game.
  • Arduino C++: I used C++ to code the Arduino's input handling and serial bus communication logic.
